PROJECT

LAW ACTS AND SECTIONS

The project entitled as “Law Acts And Sections”. Law Acts and Sections is one of the Web based application to an interactive online platform. The client can enter the problem in text area. According to the problem case details displayed in section page. This application displays the lawyer details and the sections details. It makes faster and easier to find and hire the best Lawyers in any city / court in India. Lawyers are on a mission to make the legal experience remarkable. By making-legal services high quality, cost-effective and on-demand for every need.

Languages Used

  • Frontend : HTML5, CSS3
  • Backend : Mysql 2008
  • Server side Script : PHP
  • Client side Script : JavaScript
  • Environment : Brackets, Notepad++
  • Operating System : Windows 10
